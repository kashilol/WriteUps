🔍 Hack The Box - Sherlocks Writeups

A collection of detailed writeups for Hack The Box Sherlocks — defensive cybersecurity investigation challenges focused on Digital Forensics and Incident Response (DFIR).

📖 What are Sherlocks?

Sherlocks are realistic, scenario-based investigation challenges on Hack The Box that put you in the role of a security analyst responding to an incident. Each Sherlock presents you with forensic artifacts from a compromised system and a series of guiding questions to answer - reconstructing the full attack chain from initial access to post-exploitation.
They cover areas such as:

Digital Forensics & Incident Response (DFIR)
Malware Analysis & Reverse Engineering
Threat Hunting
Log Analysis

Difficulty levels range from Easy to Insane.

📁 What's in This Repository?

Each folder contains a complete writeup for one Sherlock challenge, including:

Step-by-step methodology - how I approached the investigation
Tools used - both the commands run and why
Screenshots - evidence from the artifacts supporting each answer
Answers - clearly marked for each task
Key takeaways - what I learned from the challenge
